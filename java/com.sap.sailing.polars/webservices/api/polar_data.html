SAP Sailing Analytics Webservices API Version 1.0

URL: /api/v1/regattas/{regattaname}/races/{racename}/competitordata

Description:

Gets the binary representation of the server's polar data for all boat classes.

You need the POLAR_DATA:READ permission for this. The output can be deserialized into a Java object of type com.sap.sailing.polars.impl.PolarDataServiceImpl using its initiallyFillFromInternal method. It is the same format used by the cross-server replication of the polar data. See also the com.sap.sailing.polars.jaxrs.client.PolarDataClient class.


Webservice Type: REST
Webservice Method: GET
Output format: Binary Java Object
Example: curl -o polar.dat -H 'Authorization: Bearer {your-access-token-here}' https://www.sapsailing.com/polars/api/polar_data
